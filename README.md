## About the project

The goal of this portfolio is to provide a clear and simple presentation of:
- my web development progress  
- the projects I’ve built  
- the stack I currently use  
- ways to contact me  

The design is clean, minimal and focused on readability and structure.

---

## Features
- Responsive layout (mobile → desktop)
- Custom sections: About, Skills, Projects, Contact
- Project links with live demos and GitHub repos
- Contact section with e-mail and social links
- Clean and lightweight CSS (no frameworks)
